# SalusSGL - Sistema de Gestão de Leitos
O projeto visa promover melhorias no atendimento e organização de hospitais com um bom sistema de gestão de leitos, reduzindo a superlotação.

## Objetivo
O sistema tem como objetivo:
* Reduzir a superlotação;
* Comunicação automática entre as equipes;
* Rastreamneto de movimentações;
* Permitir o controle e monitoramento em tempo real da ocupação dos leitos;
* Melhor coordenação entre setores hospitalares.

## Tecnologias e Links Utilizados
* Linguagem PHP;
* Livewire;
* Laravel;
* HTML;
* Bootstrap;
* MySQL;
* Xampp.

## Como Utilizar Esse Repositório
Passo à passo para utlização do repositório:
1. No GitHub, copie o link de repositório;
2. No CMD, dê os comandos:
   * `git config --global user.name "nome-de-usuario"`
   * `git config --global user.email "email"`
   * `git clone https://github.com/alexfigueirinha/2026-SalusSGL.git`;
3. Entre na pasta do projeto: `cd 2026-SalusSGL`;
4. Dê o comando `code .` para abrir o projeto no VS Code.
